# 일상 가계부

 :일상에서 일어나는 다양한 일들을 기록하는 달력 어플. 사용자가 원하는 테마, 달력을 선택 가능하고 지출 및 수입 내역, 스케줄 리스트, 일기를 저장할 수 있다.
 
 
## 개발 환경  
- OS : Window 10 64bit  
- IDE : VSCode  
- Browser Support : Chrome  
- HTML , CSS, Javascript, React, TypeScript  
- Version Management : Git, Github
